# PORTIFOLIO DE DUAS APIS EM UMA,
# 🏛️ Telasco | Portal Multisite

Portal centralizado que reúne dois negócios da marca **Telasco**: um restaurante italiano e uma barbearia. O projeto é totalmente estático, construído com HTML, CSS e JavaScript puro — sem frameworks ou dependências externas.

---

## 📁 Estrutura do Projeto

```
poortifolio/
├── index.html          # Portal principal (tela de entrada)
├── portal.css          # Estilos do portal
├── portal.js           # Scripts do portal
├── assets/
│   ├── restaurant/     # Imagens do restaurante
│   └── barber/         # Imagens da barbearia
├── restaurant/
│   ├── index.html      # Site do restaurante italiano
│   ├── styles.css
│   └── script.js
└── barber/
    ├── index.html      # Site da barbearia
    ├── styles.css
    └── script.js
```

---

## 🌐 Sub-sites

### 🍝 Italiana Food (`/restaurant`)
Restaurante italiano fundado em 1985 pela família Rossi. O site apresenta:
- **Hero** com chamada para o menu
- **Menu** com especialidades: Carbonara Tradicional, Risotto ai Funghi, Tiramisù Caseiro
- **Nossa História** — apresentação institucional
- **Contato** no rodapé

### ✂️ Barbearia Telasco (`/barber`)
Barbearia clássica com ambiente moderno. O site apresenta:
- **Hero** com botão de agendamento
- **Serviços**: Corte Clássico (R$50), Barba Esculpida (R$40), Combo Imperial (R$80)
- **Galeria** de fotos
- **Agendamento** via formulário (nome, telefone, serviço)

## 🚀 Como Visualizar

Por ser um projeto estático, basta abrir o arquivo principal no navegador:

```
poortifolio/index.html
```

Ou pelo terminal:

```bash
# Windows
start msedge index.html

# macOS
open index.html

# Linux
xdg-open index.html
```
